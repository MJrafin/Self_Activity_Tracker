# Self Activity Tracker
![SAT-w1](https://user-images.githubusercontent.com/70026044/210168222-5578d9b7-b240-48b1-8f0f-0b1b4d13219e.png)


Self Activity Tracker is an office management web application that helps you to plan the activities and assists to keep track of your tasks. Collaborate with your co-workers and do productive things together!

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- Laravel
- PHP
- MySQL
